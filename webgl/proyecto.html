Sombras

There's two ways to implement centering, either on specific elements or as a global option.

renderer.shadowMap.enabled = true;

Activamos la generación de sombras en el canvas

suelo.receiveShadow = true;

Para el suelo solo activamos la posibilidad de recibir sombras con receiveShadow

cuello.receiveShadow = true;

Para todos los demás objetos activamos la posibilidad de recibir y generar sombras con receiveShadow y castShadow

Materiales

Se configuran los materiales que se van a aplicar a los diferentes objetos de la cabeza del robot teniendo en cuenta que se utiliza MeshLambertMaterial para objetos mate y MeshPhongMaterial para objetos brillantes. Estos dos tipos de materiales permiten sombras, por el contrario el tipo de material MeshBasicMaterial no permite sombras.


let materialSuelo = new THREE.MeshLambertMaterial({

Iluminación

Se aplica la iluminación al lienzo que estamos trabajando, para este caso especifico se aplica una luz ambiente con el objeto AmbientLight, una luz puntual con el objeto PointLight, una luz direccional con el objeto DirectionalLight y una luz focal con el objeto SpotLight. Después de implementar la iluminación vamos a poder ver las sombras de los objetos a los cuales les configuramos eta funcionalidad, de igual forma se pueden visualizar las texturas asignadas.


let luzAmbiente = new THREE.AmbientLight(0xFFFFFF, 0.5);

Entorno

Se implementa una textura de entorno para dar el efecto de que nuestro robot esta en un espacio abierto. Este proceso es posible utilizando las herrameintas ShaderLib.cube y ShaderMaterial en un BoxGeometry.


let walls = [path + 'Yokohama/posx.jpg', path + 'Yokohama/negx.jpg',
  path + 'Yokohama/posy.jpg', path + 'Yokohama/negy.jpg',
  path + 'Yokohama/posz.jpg', path + 'Yokohama/negz.jpg'
];

texturesD = new THREE.CubeTextureLoader().load(walls);

Se crea textura para la vista panorámica o de habitación.

let shader = THREE.ShaderLib.cube;
shader.uniforms.tCube.value = texturesD;
let shaderMaterial = new THREE.ShaderMaterial({
  fragmentShader: shader.fragmentShader,
  vertexShader: shader.vertexShader,
  uniforms: shader.uniforms,
  dephtWrite: false,
  side: THREE.BackSide
});

room = new THREE.Mesh(new THREE.BoxGeometry(500, 500, 500), shaderMaterial);
room.position.y += 250;
scene.add(room);

Vistas

Se implementa una cámara ortogonal para visualizar nuestro objeto en diferentes planos, alzado, planta y perfil con el objeto OrthographicCamera.

let l = b = -4;
let r = t = -l;

let origen = new THREE.Vector3(0,0,0);
let camaraOrthographic;
if(ar > 1){
  camaraOrthographic = new THREE.OrthographicCamera(l*ar, r*ar, t, b, -500, 500);
}
else{
  camaraOrthographic = new THREE.OrthographicCamera(l, r, t/ar, b/ar, -500, 500);
}

alzado = camaraOrthographic.clone();
alzado.position.set(0,0,100);
alzado.lookAt(origen);

perfil = camaraOrthographic.clone();
perfil.position.set(100,0,0);
perfil.lookAt(origen);

planta = camaraOrthographic.clone();
planta.position.set(0,100,0);
planta.lookAt(origen);

scene.add(alzado);
scene.add(perfil);
scene.add(planta);
